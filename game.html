Adding CSS

DANGER

If you need to add CSS code to the site add it to the hexagontheme.css file.

You must bracket your changes in the CSS file with comments like below:-

/* Stephen's CSS Changes START */
.btn-special {
  color: #563d7c;
  background-color: #fff;
  border-color: #e5e5e5;
}
/* Stephen's CSS Changes END */
